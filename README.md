# project-pizzeria

Pizzeria website with table booking and online ordering.  
This project was made as a part of Kodilla's frontend bootcamp.

Live demo [_here_](https://project-pizzeria--wojciechjohaniu.replit.app/).

## Technologies Used

- HTML5
- CSS3
- SCSS
- JavaScript (ES6+)

## Features

- Loading products data from **_json-server_** mock API
- Products ordering form with ingredients choice, its' visual representation and price calculation
- Products cart with sending order to **_json-server_** mock API
- Table booking with validation using fetch API calls to **_json-server_**

## Setup

Install project dependencies using CLI command:

> `npm install`

## Local environment

Start local server:

> `npm run watch`

Create production build:

> `npm run build`

Validate HTML and lint JavaScript, SCSS files:

> `npm test`
